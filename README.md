## 📌 Overview

This project presents an executive-level **Power BI dashboard** analyzing global retail sales performance across multiple regions, product categories, and time periods.

Built to demonstrate how business intelligence tools can transform **100,000+ raw records** into clear, actionable insights that drive real business decisions — from regional strategy to inventory planning.

## 💡 Key Insights

- **West region leads** total revenue at approximately $0.7M, indicating strong market penetration and customer demand
- **Seasonal sales spikes** detected across multiple periods, suggesting promotional campaign impact or demand shifts
- **Top product concentration** — a small group of high-performing products drives a disproportionate share of total revenue
- **Anomaly detection** surfaces unusual sales activity, flagging potential operational disruptions or promotional events for further investigation

---

## ⚙️ How It Works

```
Raw Data Sources → Python ETL Pipeline → Power Query Transformation
    → DAX Measures → Power BI Dashboard → Executive Insights
```

1. **Extract** — Pull data from multiple sources using Python and SQL
2. **Transform** — Clean, normalize, and reshape using Power Query (M) and Pandas
3. **Model** — Build Star Schema data model for optimized performance
4. **Measure** — Write DAX measures for KPIs, period-over-period comparisons, and trend analysis
5. **Visualize** — Design interactive Power BI dashboard with drill-down capability
6. **Deliver** — Executive-ready insights with regional and time-based filtering

---
